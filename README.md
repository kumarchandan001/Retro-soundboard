# Retro Soundboard

A nostalgic retro Indian TV soundboard built with plain HTML, CSS, and JavaScript. The app lets you play classic theme songs and famous TV dialogues from a browser-based soundboard interface.

## Features

- Search sounds in real time
- Control global volume with a slider
- Stop all currently playing sounds at once
- See the current track in a now-playing display
- Play retro theme songs, dialogues, and sound effects

## Project Structure

- index.html - page structure and controls
- style.css - retro-themed styling and responsive layout
- script.js - sound playback, filtering, and UI behavior
- assets/ - audio files used by the soundboard

## How to Run

1. Clone or download this repository.
2. Open index.html in your browser.
3. Click any button to play a sound.

If you are using a local editor like VS Code, you can also open the folder and launch index.html with Live Server.

## Sounds Included

The soundboard currently includes clips such as:

- Malgudi Days
- Mahabharat
- Byomkesh Bakshi
- Jungle Book
- Shaktimaan
- Mogambo Khush Hua
- Doordarshan Tune

## Notes

- All audio files are stored inside the assets/ folder.
- The soundboard stops any currently playing clip before starting a new one.
- The search box filters buttons by their visible text.
